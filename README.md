# Тестирование REST API в Postman

## О проекте
В этом репозитории представлены автотесты для API-тестирования.

## Содержимое
- `QamoveUp.postman_collection.json` — коллекция запросов для тестирования REST API
- `QamoveUp.postman_environment.json` — файл окружения с переменными

## Что тестируется
- Авторизация (получение токена)
- CRUD операции с пользователями
- Проверка статус-кодов и структуры JSON-ответов
- Негативные сценарии (ошибки 400, 403, 404, 422, 429)

## Используемые технологии
- Postman
- JavaScript (pm.test)
- REST API
